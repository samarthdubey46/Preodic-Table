# 🧪 Periodic Table Memory Trainer

An interactive web app designed to help you practice and test your knowledge of the periodic table.

Whether you're preparing for chemistry exams, revising element symbols, or just trying to prove that years of science classes weren't completely wasted, this tool helps you strengthen your periodic table recall through active practice.

## ✨ Features

- Interactive periodic table
- Practice recalling element names and symbols
- Improve memory through repeated testing
- Simple and responsive user interface
- Fast and lightweight web application
- Suitable for students and chemistry enthusiasts

## 🎯 Purpose

Most students memorize the periodic table once and then immediately donate that knowledge to the void.

This project was built to make periodic table revision more engaging by allowing users to actively practice and measure how much they remember rather than passively reading a chart.
